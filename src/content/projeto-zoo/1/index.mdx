# Visão Geral

## Descrição

Aplicativo de gestão para zoológicos, com controle de animais, de procedimentos, de recintos, da equipe, de visitantes e com dashboard e relatórios.
O sistema foi desenhado para funcionar Offline First, permitindo operação em locais remotos sem conectividade constante, com sincronização automática assim que a conexão for restabelecida.

## Tecnologias

- Banco de Dados: PostgreSQL
- Back-end: NestJS + Prisma ORM
- Front-end Mobile: React Native
- Comunicação Assíncrona: RabbitMQ
- Hospedagem do servidor: Hostinger VPS
- Sincronização offline/online: Firebase Firestore

## Modelo arquitetural

- Arquitetura baseada em DDD (Domain-Driven Design) com separação clara dos contextos de negócio e comunicação baseada em eventos
- Multi-tenant, garantindo separação lógica de dados e segurança entre as organizações, permitindo escalabilidade horizontal futura
