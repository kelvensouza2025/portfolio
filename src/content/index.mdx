# Bem-vindo

Este site reúne a documentação técnica de projetos desenvolvidos com foco em boas práticas de engenharia de software, arquitetura de sistemas, e design orientado à escalabilidade.

## Sobre

Sou desenvolvedor pleno e arquiteto de software em formação, com experiência prática em:

- Prototipação e design system (Figma)
- Padrões de projeto (SOLID, Clean Code, MVC)
- DevOps e Automação (Docker, GitHub Actions)
- Deploy e operação em ambientes cloud (AWS, Vercel)
- Desenvolvimento back-end (TypeScript, NestJS, Laravel, Prisma ORM)
- Desenvolvimento front-end e mobile (TypeScript, Vue.js, Next.js, React Native)

Meu objetivo é construir soluções que sejam escaláveis, confiáveis e fáceis de manter.
