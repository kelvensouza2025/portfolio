# Pontos de Evolução

- Implementação de login social
- Offline first no aplicativo mobile
- Painéis dinâmicos com gráficos interativos
- Implementação de auditoria de eventos (log de sincronizações, falhas, retries)
- Ampliação da integração com ERP Bling para sincronização bidirecional de produtos
- Migração planejada da arquitetura MVC para Arquitetura em Camadas, visando melhor separação de responsabilidades e escalabilidade
