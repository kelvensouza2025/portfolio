# Visão Geral

## Descrição

Sistema de gerenciamento comercial, permitindo cadastro de usuários, de clientes, de produtos, de serviços, de orçamentos, de ordens de serviço, de comissões e visualização de dashboard e relatórios. Além da versão web, também será desenvolvido um aplicativo mobile para iOS e Android com funcionalidades similares e notificações em tempo real.

## Tecnologias

- Banco de dados: PostgreSQL
- Back-end: Nest.js + Prisma ORM
- Mobile App: React Native + Expo
- Armazenamento de imagens: AWS S3
- Front-end Web: Next.js + Tailwind CSS
- Hospedagem: Vercel (Front-end) e AWS Elastic Beanstalk (Back-end)

## Modelo arquitetural

- MVC (Model-View-Controller)
- Multi-tenant, garantindo separação lógica de dados e segurança entre as organizações, permitindo escalabilidade horizontal futura
